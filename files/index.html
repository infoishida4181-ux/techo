組み込み祝日（自動生成）
日本の祝日は年ごとに自動生成されます（2023〜2040年対応）。
同じ日付でカスタム祝日を追加するとそちらが優先されます。

リマインダーを取り込む
取り込み方法
1
iPhoneのショートカットアプリで、リマインダーを取得してJSONに変換するショートカットを作成します
2
ショートカットを実行すると、JSONテキストがクリップボードにコピーされます
3
下の入力欄に長押し→ペーストしてから「読み込む」をタップしてください
4
取り込んだリマインダーはTask画面に表示され、期限付きのものはカレンダーにも反映されます
JSONデータを貼り付け
JSON形式サンプル（ショートカット設計用）
[ { "title": "朝礼", "dueDate": "2026-04-07", "dueTime": "08:45", "allDay": false, "notes": "", "completed": false, "list": "仕事", "priority": 1, "source": "reminder" } ]

i

→ 繰り返し で各項目を辞書に変換
→ テキストに変換（JSON形式）
→ クリップボードにコピー
